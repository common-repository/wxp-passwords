Protect your wordpress content by passwords.

== Description ==

1. Create multiple passwords for specified time.
2. Password with time will expire after specified time and content will be locked again.
3. If no time selected in password then password will be always valid.
4. You can create multiple passwords with different content access.
5. Just share password with user and he will be able to access allowed content.

== Installation ==

1. Simply install and activate the plugin.
2. Create passwords with rules and lock your content.
3. For more details please check installation video.
